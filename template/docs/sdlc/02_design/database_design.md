# Database Design

## Engine
- **Primary**: {{Postgres 16}}
- **Reason**: {{ACID + JSONB support, mature ops}}
- **Encoding**: UTF-8, timezone UTC

## Conventions

- Table names: `snake_case`, plural (`users`, `orders`)
- PKs: `id BIGINT GENERATED BY DEFAULT AS IDENTITY` or `uuid v7`
- Timestamps: `created_at`, `updated_at`, `deleted_at` (soft delete if needed) — all `TIMESTAMPTZ`
- Foreign keys: always `ON DELETE` explicit (`RESTRICT`/`CASCADE`/`SET NULL`)
- Index every FK unless justified
- Money: `NUMERIC(19,4)` — never `FLOAT`
